# User stories

### Tancerz

1. Jako **_tancerz_** mogę **_się zalogować_**, aby **_mieć dostęp do swoich danych_**.
2. Jako **_tancerz_** mogę **_wyświetlić swoje wypożyczenia, zwroty i zagubienia_**, aby **_sprawdzić co i kiedy muszę zwrócić_**.
3. Jako **_tancerz_** mogę **_edytować swoje dane kontaktowe (email i telefon)_**, aby **_były aktualne_**.
4. Jako **_tancerz_** mogę **_zmienić swoje hasło_**, aby **_zwiększyć bezpieczeństwo_**.
5. Jako **_tancerz_** mogę **_zresetować hasło_**, aby **_odzyskać dostęp do konta_**.

### Kierownik (dodatkowo)

1. Jako **_kierownik_** mogę **_rejestrować wypożyczenia, zwroty i zagubienia elementów garderoby_**, aby **_prowadzić ewidencję_**.
2. Jako **_kierownik_** mogę **_zarządzać elementami garderoby_**, aby **_stan garderoby był aktualny_**.
3. Jako **_kierownik_** mogę **_kompletować elementy garderoby w stroje_**, aby **_ułatwić wypożyczanie_**.

### Administrator (dodatkowo)

1. Jako **_administrator_** mogę **_zarządzać użytkownikami_**, aby **_kontrolować ich dostęp do systemu_**.
