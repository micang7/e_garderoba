# E-garderoba - backend

Aplikacja serwerowa oparta o **NestJS** i **PostgreSQL**.

## Instalacja

```bash
npm install
```

## Uruchomienie

**Tryb developerski:**

```bash
npm run start:dev
```

**Tryb debuggowania:**

```bash
npm run start:debug
```

**Tryb produkcyjny:**

```bash
npm run build
npm run start:prod
```

## Testy
```bash
npm test
npm run test:e2e
```
