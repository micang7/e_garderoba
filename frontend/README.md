# E-garderoba - frontend

Aplikacja kliencka oparta o **React + Vite**.

## Instalacja

```bash
npm install
```

## Uruchomienie

**Tryb developerski:**

```bash
npm run start:dev
```

**Tryb produkcyjny:**

```bash
npm run build
npm run start:prod
```

## Testy

```bash
npm run test
npm run test:e2e
```
